# 🎮 Pac-Man Game

Arcade-style Pac-Man game developed using JavaScript, HTML, and CSS.  
This project recreates the classic Pac-Man gameplay, including movement, collision detection, and scoring mechanics.

---

## 🚀 Features

- Player movement with keyboard controls
- Basic enemy (ghost) behavior
- Collision detection with walls
- Score system based on collected items
- Game loop and real-time updates
- Browser-playable

---

## 🛠️ Technologies Used

- JavaScript (Game logic)
- HTML5 (Structure)
- CSS3 (Styling)

---

## 🎮 How to Play

- Use the arrow keys to move Pac-Man:
  - ⬆️ Up  
  - ⬇️ Down  
  - ⬅️ Left  
  - ➡️ Right  
- Eat all the dots to increase your score  
- Avoid ghosts or you lose the game
